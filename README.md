# Simple Contact & Task Manager

## Tech Stack
- PHP
- MySQL
- HTML & CSS
- XAMPP

## Features
- Admin Login (hardcoded)
- Contact Management
- Task Assignment
- Task Status (Pending / Done)
- Session-based access control

## How to Run
1. Place folder in `htdocs`
2. Start Apache & MySQL in XAMPP
3. Create database `contact_task_db`
4. Import `database.sql` via phpMyAdmin
5. Open `http://localhost/contact_task_manager/login.php`
